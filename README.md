# Employee Management System (Flask CRUD Application)

A web-based CRUD application built with **Flask**, **SQLAlchemy**, and **Bootstrap** for managing employee records.  
The system allows users to create, read, update, and delete employee data in a clean and responsive interface.

---

## ✨ Features
- **Create** – Add new employees with name, email, and phone details.  
- **Read** – View all employees in a styled Bootstrap table.  
- **Update** – Edit employee details through modal forms.  
- **Delete** – Remove employees with confirmation prompts.  
- **Flash Messages** – User-friendly feedback for each action (add, update, delete).  
- **Database** – Uses SQLite by default (easily switchable to MySQL/PostgreSQL).  

---

## 🛠️ Tech Stack
- **Backend:** Flask (Python)  
- **Database:** SQLAlchemy ORM with SQLite  
- **Frontend:** HTML, Bootstrap, Jinja2 Templates  
- **Other:** Flask Flash for messages
